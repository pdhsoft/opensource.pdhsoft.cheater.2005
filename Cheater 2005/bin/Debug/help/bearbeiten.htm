PDHSoft Cheater 2005 | Funktionen - Bearbeiten

So Bearbeiten Sie Cheats in der Datenbank. Klicken Sie auf Cheat Bearbeiten im Menü Datei oder das Sysmbol in der Symbolleiste. Es Öffnet sich das Fenster von Bild 1. Wählen Sie einen System Typ und gg ein System und eine Kategorie, Klicken Sie auf den Button "Cheat bearbeiten" im sich öffnendem Fenster wählen Sie die gewünschte Datei aus und klicken dort auf öffnen.

Es öffnet sich nun folgendes Fenster von Bild 2. Unten im weißen Feld können Sie den Cheattext Bearbeiten. Klicken Sie nach ihrer Änderung auf den Button "Speichern...".
