PDHSoft Cheater 2005 | Funktionen - Erstellen

So erstellen Sie einen Cheat, Klicken SIe auf Cheat erstellen im Menü Datei oder auf das Sysmbol in der Symbolleiste. Es Öffnet sich das Fenster von Bild 1. Wählen Sie einen System Typ, wählen Sie g.g. ein System und wählen Sie eine Kategorie, klicken Sie auf Cheat hinzufügen.

Bei Cheats, Codes, Lösungen & Tipps öffnet sich das Fenster von Bild 2. Geben Sie hier einen Spiele Titel ein -( darf folgende Zeichen nicht enthalten: \ /:*"?<>| )- geben Sie einen Author ein -( darf folgende Zeichen nicht enthalten: \ /:*"?<>| )- Ändern Sie g.g. das Datum über den Button Datum ändern... Geben Sie einen Cheat Text ein und klicken Sie auf den Button "Speichern...".

Bei Grafiken, Tools, Trainer, Savegames, Patches & Updates erscheint das Fenster von Bild 3. Geben Sie hier einen Spiele Titel ein -( darf folgende Zeichen nicht enthalten: \ /:*"?<>| )- geben Sie einen Author ein -( darf folgende Zeichen nicht enthalten: \ /:*"?<>| )- Geben Sie eine Zusaätzliche Bezeichnung ein -( darf folgende Zeichen nicht enthalten: \ /:*"?<>| )- , Wählen Sie einen Dateitypen und geben Sie ein Datum ein. Es sind nur folgende Dateitypen erlaubt. *.exe,*.zip,*.msi,*.jpg,*.gif,*.wmf,*.bmp. Gepackte Dateien dürfen NICHT Verschlüsselt sein (KEIN PASSWORT). Alle Dateien müsen Viren, Trojaner, Würmer, Spyware etc... Frei sein, Verstöße werden Angezeigt. Klicken Sie nun auf den Button "Datei der Datenbank hinzufügen".

Tipps zum Erstellen

Sie können z.B. Dem Spiele Titel auch noch ein Kürzel angeben z.B. für (Need for Speed das Kürzel NFS) so muß bei der Cheatsuche nicht der volle Text geschrieben werden.
