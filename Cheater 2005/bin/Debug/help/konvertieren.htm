PDHSoft Cheater 2005 | Funktionen - Konvertieren

So können Sie PDHSoft Patte's Cheater 2004 Cheatdateien Konvertieren. Klicken Sie auf Cheat Konvertieren im Menü Datei oder auf das Symbol in der Symbolleiste. Es Öffnet sich ein Fenster, in diesem Fenster öffnen Sie die *.cheat Datei, des Cheats, den Sie Konvertieren möchten, klicken Sie auf den Button "Öffnen". Es öffnet sich nun Folgendes Fenster (Bild 1) .

Methode 1:

Klicken SIe auf den Button "Konvertieren" . Geben Sie den Spieletitel und Author ein -( darf folgende Zeichen nicht enthalten: \ /:*"?<>| )- Ändern Sie gg Das Datum über den Button "Datum Ändern" Wählen Sie einen System Typ, gg ein System und eine Kategorie aus. Bearbeiten Sie nun den Cheat Text (Entfernen Sie SpieleTitel, System und Author aus dem Cheattext) klicken Sie anschließend auf den Button "Speichern..."

Methode 2:

Geben Sie den Spieletitel und Author ein -( darf folgende Zeichen nicht enthalten: \ /:*"?<>| )- Ändern Sie gg Das Datum über den Button "Datum Ändern" Wählen Sie einen System Typ, gg ein System und eine Kategorie aus. Klicken Sie in das Untere Feld und drücken die Tasten Kombination "STRG+ C" klicken Sie nun mit der rechten Maustaste in das obere textfeld und wählen den Eintrag Einfügen. Bearbeiten Sie nun den Cheat Text (Entfernen Sie SpieleTitel, System und Author aus dem Cheattext) klicken Sie anschließend auf den Button "Speichern..."
